Конвертер валют на Python

Описание проекта

Это простая программа на Python для конвертации валют.  
Пользователь вводит сумму и выбирает направление перевода, после чего программа выводит результат.

Цель проекта

- Научиться работать с функциями
- Закрепить условия (`if`, `elif`)
- Использовать циклы (`while`)
- Реализовать меню программы
- Практиковаться с числами (`float`)

Функционал программы

Программа поддерживает следующие операции:
- USD → KGS
- KGS → USD
- EUR → KGS
- KGS → EUR

Также есть:
- меню выбора
- возможность выхода из программы

Как работает программа

1. Пользователь выбирает действие из меню  
2. Вводит сумму  
3. Программа вызывает нужную функцию  
4. Выводится результат  

Используемые функции

convert_usd_to_kgs(amount)
Переводит доллары в сомы

convert_kgs_to_usd(amount)
Переводит сомы в доллары

convert_eur_to_kgs(amount)
Переводит евро в сомы

convert_kgs_to_eur(amount)
Переводит сомы в евро
